# Симулятор дальтонизма

Интерактив: как выглядит интерфейс глазами человека с нарушением цветовосприятия.

**Живая страница:** https://viktoriafilippova.github.io/daltonism-simulator/

## Что умеет

- Четыре типа: дейтеранопия, протанопия, тританопия, ахроматопсия
- Ползунок тяжести — от лёгкой формы до полной. У большинства людей потеря
  частичная, и разница между 30% и 100% сильно меняет картину
- Готовые сценарии: статусы заказа, график, схема линий метро, фото
- У каждого сценария есть исправленная версия — как дизайн чинится вторым
  признаком помимо цвета
- Загрузка своей картинки: можно проверить собственный макет

## Как это считается

Матрицы Machado, Oliveira, Fernandes (2009) — стандарт для симуляции
цветовосприятия. Это приближение: дальтонизм у каждого человека свой,
и точную картину даёт только живой человек. Симулятор — способ увидеть
проблему, а не заменить тестирование.

## Зачем

Красный и зелёный — самая частая пара для статусов «ошибка/успех».
Для 8% мужчин эта пара не работает. Правило простое: цвет никогда
не единственный носитель смысла (WCAG 1.4.1).
